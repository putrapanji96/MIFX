# MIFX

## 1. run this to install all dependencies 
```sh
$ yarn install 
```
## 2. pod install for IOS
```sh
$ cd ios && pod install && cd ..
```
## 3. create file .env.dev 
```dosini
BASE_URL=https://api.example.org
```
 ## 4. To start Metro
 ```sh
$ yarn run start
```
## 5.run this to run the app
(Android)
 ```sh
$ yarn run android
```
(IOS)
 ```sh
$ yarn run ios
```
